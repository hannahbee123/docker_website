Basic Linux Commands

  • pwd tells you which directory you are currently in

  • cd change directory (enter the name of the directory you want to go to after ‘cd’)

  • cd / move to the root directory

  • cd .. to go back one folder from the directory you’re currently in (the two dots represent ‘back’)

  • cd ~ brings you back to your home directory

  • ls tells you what files are in the directory you’re currently in

  • ls -a tells you what files are in the directory you’re currently in, including hidden files

  • mkdir makes a directory (enter the name of the new directory after ‘mkdir’)

  • rm removes both the folder and the files it contains (enter the name of the directory to be removed after ‘rm’)

  • man shows more about a particular command and how to use it (enter the name of the command you want to learn more about after ‘man’)

  • --help shows which ways a particular command can be used (enter the name of the command you want to learn more about after ‘--help’)
